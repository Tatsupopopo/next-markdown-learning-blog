## 変更した部分

Markdown ファイルを remark で変換した後に, `className="prose-custom"` の属性が付いたタグでラップしています. したがって, その子要素のタグに CSS を当てることで Markdown から変換した HTML のみに装飾を施しています.

### 文字サイズの変更

文字サイズを `text-base` から `text-sm` に変更しています. 

アクセシビリティの基準ギリギリの小ささ（14pt）なんですが, スマホで見たときにちょっと大きいと感じたことと, 特段高齢者をターゲットにしているわけではないのでこのフォントサイズで落ち着きました.

### h2, h3 タグの装飾

h2 タグに下線を, h3 タグに左の装飾を付けました.

h2 の見出しではセクション自体を分けて読んでもらう意図で横に区切る線を, h3 の見出しでは見出しを含めて文章を読んでもらう意図で区切らずポイントとなるような装飾を付けています.

具体的には CSS プロパティは以下を設定しました.

```css
/* Tailwind.css */
h2 {@apply border-b border-slate-200;}
h3 {@apply border-l-4 border-slate-200;}
```

### リンクテキストの装飾

よくある青字のリンクテキストに, ホバーでハイライトされるようにしました. 多分素の HTML だとデフォルトで似たような感じになっているので, 後付けで子の装飾をするのは新鮮でした.

```css
/* Tailwind.css */
a {@apply text-sky-600 underline underline-offset-2 decoration-sky-300;}
a:hover {@apply text-sky-700 decoration-sky-400;}
```
